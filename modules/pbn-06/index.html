VEYRA PBN PATH · MÓDULO 06

Waypoints y terminadores de trayectoria

Una línea en la carta es una instrucción geométrica que la base de datos traduce para el FMS. Comprende cómo se define cada tramo, cuándo anticipa el giro y por qué no todas las aeronaves pueden ejecutar todas las trayectorias.

Duración estimada: 30–38 minutos. Incluye fly-by/fly-over, concepto path & terminator, IF, TF, CF, DF, RF y otras terminaciones operacionales.
01 · DEL PROCEDIMIENTO AL FMS

La carta comunica; la base de datos codifica

La descripción publicada indica qué debe volarse. En la base de datos, cada tramo se representa mediante un tipo de leg: una trayectoria y el evento o punto que la termina.

PATH

Cómo llegar

Puede ser un track, curso, rumbo, directo o arco de radio constante.

TERMINATOR

Hasta cuándo

Un fix, una altitud, una distancia, una interceptación u otra condición finaliza el tramo.

LEG

La instrucción completa

El sistema combina ambas ideas para construir una trayectoria repetible.

Trayectoria + terminación = tramo codificadoEl leg type no es una autorización ATC ni una NavSpec; es la forma en que se representa un segmento.
02 · QUÉ ES UN WAYPOINT

Una posición definida, no necesariamente una radioayuda

Un waypoint puede definirse por coordenadas, por relación con ayudas terrestres o mediante otros criterios aprobados. Sirve para organizar rutas, transiciones, giros y restricciones.

FIX NOMBRADO

Identidad publicada

Puede aparecer en carta y base de datos con un nombre de cinco letras u otra identificación aplicable.

WAYPOINT NO PUBLICADO

Construcción interna

Algunos puntos ayudan a definir una trayectoria codificada sin requerir que la tripulación los trate como un fix operativo independiente.

No crear puntos manuales para “reparar” un procedimiento.

La modificación de procedimientos depende de la autorización, la capacidad y las políticas aplicables. Una geometría visualmente parecida no garantiza la protección prevista.

03 · FLY-BY

El giro se anticipa para interceptar el siguiente tramo

FUNCIÓN

Transición suave

El sistema inicia el cambio de trayectoria antes de alcanzar el waypoint para incorporarse al siguiente leg.

Depende de: velocidad, ángulo de giro, viento, capacidad del sistema y límites de banco. La aeronave no “corta” libremente: sigue una lógica aprobada de transición.
04 · FLY-OVER

Primero cruza el waypoint; después gira

FUNCIÓN

Sobrevuelo requerido

La aeronave debe alcanzar el waypoint antes de iniciar la transición al siguiente tramo.

Puede producir mayor desplazamiento exterior.

La geometría del giro difiere del fly-by. La tripulación debe reconocer la simbología y anticipar carga de trabajo, velocidad y dirección de salida.

05 · IF Y TF

El comienzo lógico y el tramo recto principal

LegSignificadoInterpretación útil
IF · Initial FixEstablece un fix inicial para comenzar una secuencia; por sí mismo no define una trayectoria.Es el punto de partida lógico para el leg que sigue. No confundir el código IF con la denominación operacional “Intermediate Fix”.
TF · Track to FixTrayectoria geodésica entre dos fixes; el sistema corrige viento para mantener el track.Es el tramo recto más común de una secuencia RNAV.
Lectura mental: IF prepara el inicio; TF dice “mantén esta trayectoria definida hasta el siguiente fix”.
06 · CF Y DF

Mismo destino, distinta forma de llegar

CF · COURSE TO FIX

Curso definido hasta un fix

El tramo termina en un fix, pero la trayectoria está determinada por un curso especificado. Puede requerir interceptar y mantener ese curso.

DF · DIRECT TO FIX

Desde posición actual o no especificada

Construye una trayectoria geodésica directa hacia un fix desde la posición presente o desde un origen no definido previamente.

“Directo a” puede alterar la geometría prevista.

Activar un direct-to no equivale siempre a reanudar el procedimiento tal como fue publicado. Deben considerarse autorización ATC, interceptación, restricciones y política operacional.

07 · RF

Un arco de radio constante hasta un fix

Radius to Fix define una trayectoria circular alrededor de un centro calculado, desde un fix inicial hasta un fix final. Permite curvas precisas y repetibles, pero exige capacidad específica.

GEOMETRÍA

Radio constante

El arco está ligado a un centro y termina en un fix; no es simplemente “girar hacia el siguiente punto”.

CAPACIDAD

Función RF aprobada

No todos los FMS ni todas las aprobaciones soportan RF. La carta identifica cuándo se requiere.

EJECUCIÓN

Velocidad y guía

Respetar límites publicados y mantener acoplada o guiada la trayectoria según procedimientos es esencial para la protección.

Distinción: una curva fly-by suaviza la unión de dos legs; un RF leg es, en sí mismo, un tramo curvo definido.
08 · TERMINADORES SIN FIX

No todos los tramos terminan en un waypoint

Algunos legs finalizan cuando se alcanza una condición. Conviene reconocerlos porque explican por qué el FMS no muestra siempre una línea “de fix a fix”.

CA

Course to Altitude

Volar un curso especificado hasta alcanzar una altitud.

VA

Heading to Altitude

Volar un rumbo hasta alcanzar una altitud.

CI / VI

To Intercept

Curso o rumbo hasta interceptar el tramo definido que sigue.

HM

Holding

Espera codificada con terminación manual u otra lógica aplicable.

Curso y rumbo no son sinónimos: un curso describe una trayectoria respecto al terreno; un rumbo describe hacia dónde apunta la aeronave. El viento puede separarlos.

11 · ESCENARIO OPERACIONAL

ATC autoriza directo a un fix intermedio

La aeronave vuela una llegada con restricciones. ATC autoriza “direct FIX B”. ¿Qué acción refleja mejor el modelo mental correcto?

Toca una opción. El cambio de leg puede cambiar la geometría y las restricciones aplicables.
